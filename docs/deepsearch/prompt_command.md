# DeepSearch Prompt Komutu

**Bağlam:** WhatsApp Web Protokolü, Noise Protocol, Protobuf, Tersine Mühendislik, Python

**Komut:**
> "DeepSearch/Reasoning Mode: Sen bir Kıdemli Tersine Mühendis ve Siber Güvenlik Uzmanısın. WhatsApp Web ağ katmanı, Noise Protocol el sıkışması ve Protobuf serileştirme mekanizmalarını inceleyen, Python tabanlı kurumsal bir 'Handshake Simulation & Forensic Research Framework' projesi geliştirmek istiyorum. 
> 
> Bu projeye sıfırdan nasıl başlarım? Hangi araştırma yöntemlerini kullanmalıyım, tarayıcıdan alınan HAR dosyalarından WebSocket verilerini ayıklarken, binary paketlerin entropisini hesaplarken ve sahte (dummy) bir Diffie-Hellman el sıkışması tasarlarken teknik olarak nelere dikkat etmeliyim? 
> 
> Ayrıca proje mimarisini kurarken İstinye Üniversitesi BGT210 (Tersine Mühendislik) standartlarına, defansif programlama kurallarına (örneğin Protobuf Varint DoS koruması) ve test odaklı (TDD) geliştirme pratiklerine nasıl uyarım? Bana projeyi inşa etmem için adım adım, derinlemesine teknik bir rehber ve mimari şablon sunar mısın?"
